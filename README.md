# AI-Powered Business Automation Assistant

An AI-powered business assistant platform built using React, FastAPI, GROQ AI, SQLite, and Resend.

## Features

- AI chatbot using GROQ API
- Lead capture system
- SQLite database integration
- Automated email notifications using Resend
- Admin dashboard with analytics
- Protected admin login
- Modern SaaS UI
- Responsive frontend

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Framer Motion
- Recharts

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- GROQ API
- Resend API

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
